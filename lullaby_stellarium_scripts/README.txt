Not technically a *python* animation, but it feels like it belongs as part of this repo...

Step 1: run Stellarium with input script model_runnable.js to create screen shots
Step 2: run make_video_script to combine screen shots, subtitles and audio

Subtitle format is "Advanced Substation Alpha" (file extension .ass)
Documentation at
  https://hhsprings.bitbucket.io/docs/programming/examples/ffmpeg/subtitle/
  http://docs.aegisub.org/3.2/ASS_Tags/
  http://moodub.free.fr/video/ass-specs.doc
